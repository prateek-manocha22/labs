# 🏥 Healthcare Agent Starter Kit

Build an agentic AI system for real-time patient health monitoring.

## 🚀 Contents
- `app.py`: Simple dashboard structure (FastAPI).
- `data/sample_vitals.json`: Sample patient stream for testing.
- `requirements.txt`: Kit-specific dependencies.

## 🛠️ Getting Started
1. Install dependencies: `pip install -r requirements.txt`
2. Run the dashboard: `python app.py`

## 🧠 Approach
1. **Data Ingestion**: Monitor vital signs in real-time.
2. **Anomaly Engine**: Detect life-threatening drifts in BP, Heart Rate, and Oxygen.
3. **Agentic Action**: Autonomously alert doctors or suggest interventions.
