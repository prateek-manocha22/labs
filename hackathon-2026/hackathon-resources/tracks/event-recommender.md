# 🎫 Event Recommender

## 📅 Overview
Create a personalized event recommendation engine using collaborative filtering and user context to surface relevant local and online events.

- **Expected Output**: API or UI that delivers personalized recommendations with reasoning.
- **Why it Matters**: Poorly targeted events waste time. Good ones drive real community value.
