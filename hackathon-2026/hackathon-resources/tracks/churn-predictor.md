# 🌱 Churn Predictor (Foundation)

## 📉 Overview
Build a customer churn prediction model with explainable outputs, actionable retention recommendations, and a simple front-end.

- **Expected Output**: Classification model + feature importance + retention suggestions UI.
- **Why it Matters**: Retaining one customer is 5x cheaper than acquiring a new one.
